# ГАК НВ Dashboard

Компактный статический дашборд рекламных показателей по месяцам.

## Что внутри

- выбор и сравнение месяцев разных лет;
- KPI по показам, кликам, CTR, расходу, конверсиям и CPA;
- графики для сравнения 2+ месяцев;
- при выборе одного месяца графики скрываются;
- кампании сгруппированы по месяцам и раскрываются отдельно;
- мобильная версия с компактными KPI и карточным видом кампаний.
